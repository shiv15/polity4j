# Security Policy

## Supported Versions

We actively provide security support for the following versions of Polity4j:

| Version | Supported |
| --- | --- |
| 1.0-SNAPSHOT | :white_check_mark: |
| < 1.0 | :x: |

---

## Reporting a Vulnerability

If you identify a security vulnerability within Polity4j, please do **not** open a public issue. Instead, report it responsibly by following these steps:

1. Report it privately via the repository's GitHub Security tab (or contact the maintainers directly).
2. Include details about the vulnerability, potential impacts, and steps or sample code to reproduce.
3. We will acknowledge receipt of your report within 48 hours and work with you to coordinate a release and advisory.
